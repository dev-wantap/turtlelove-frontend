## 1. 회원가입/로그인 & 인증

- 대학 이메일 인증
- `ac.kr`, `edu` 두 도메인만 허용
- 아이디=대학메일, PW 로그인 방식
- 닉네임 익명 방식 (에브리타임)
- 성별 정보는 수집, 미공개

## 2. 게시글 (상담 요청)

- 카테고리 분류
    - 다음 회의때 논의 필요
- 공개 범위
    - 전체 공개 / 같은 대학교에는 숨기기
    - 전체 공개 / 특정 성별에게만 공개

## 3. 댓글 (답변)

- 단일 계층 구조 (대댓글 불가)
- AI 활용 비정상 답변 필터링

## 4. 1:1 채팅

- 단방향 시작 (글쓴이만 댓글 작성자에게 채팅 시도 가능)
- 게시글 ID, 댓글 ID를 연결하여 어느 게시글에 대한 대화인지 컨텍스트 유지
