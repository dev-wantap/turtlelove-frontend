### 행동 주체

- 비회원 : 로그인 X
- 회원 : 대학 이메일 인증 및 로그인 완료
- 관리자

---

# 1. 회원 관리

|UseCase ID|UseCase 명|주요 액터|설명|
|---|---|---|---|
|**UC-01**|회원가입|비회원|`.ac.kr`, `.edu` 이메일로 본인 인증 후 PW, 닉네임, 성별 정보 입력|
|**UC-02**|로그인|비회원|이메일(아이디)과 비밀번호로 로그인|
|**UC-03**|로그아웃|회원|현재 세션 종료|
|**UC-04**|내 정보 조회|회원|마이페이지에서 내가 쓴 글, 댓글, 채팅 목록 확인|
|UC-01||||

1. 비회원 사용자가 회원가입 버튼을 클릭
2. 시스템이 회원가입 페이지로 이동
3. 이메일(.ac.kr / .edu) 이용 대학생 인증
    1. 이메일 API 이용 OTP 발송
    2. 사용자가 시스템에 OTP 입력
    3. 시스템이 일치 여부 검증
4. 통과 시, 이메일은 고정/ PW, 닉네임, 성별 입력
5. 회원가입 완료

# 2. 게시글

|UseCase ID|UseCase 명|주요 액터|설명|
|---|---|---|---|
|**UC-05**|게시글 목록 조회|비회원, 회원|카테고리별/정렬(최신/인기)별 게시글 리스트 확인|
|**UC-06**|게시글 상세 조회|회원|특정 게시글의 내용과 댓글 확인 (성별/공개범위에 따라 노출 제어)|
|**UC-07**|게시글 작성|회원|제목, 내용, 카테고리, **공개 범위(대학/성별 필터링)** 선택하여 등록|
|**UC-08**|게시글 수정/삭제|회원|본인이 작성한 게시글 내용 변경 또는 삭제|

UC-07

1. 작성하고자 하는 카테고리에 진입 (UC-05)
2. 사용자(회원)이 게시글 작성 버튼 클릭
3. 작성 페이지 이동 : 제목, 본문, 공개 범위(같은 대학 / 성별 필터링)
4. 사용자(회원)이 작성 완료 버튼 클릭
5. 게시글 등록 및 게시

UC-08 (From 마이페이지)

1. UC-06에서 연결
2. 본인 글이면 수정 버튼 표시
3. 수정 버튼 클릭
4. 작성 페이지 이동 및 내용 채워주기
5. 내용 수정
6. 수정 완료 버튼 클릭
7. 수정 완료
